## Features

- **Extremely Fast**: Built on top of a highly optimized Electron and Vencord stack, providing insanely fast startup times and snappier UI interactions.
- **Privacy First**: Complete control over your Discord client data, built-in tracking protection, and OS-level privacy screen to hide your app from streaming software.
- **Built-in Plugins**: Ships natively with powerful, exclusive plugins including instant screensharing, hidden friends toggles, and UI enhancements.
- **Vencord Compatibility**: Full native support for Vencord plugins and themes out of the box.

## Installation

Grab the latest installer from the [Releases page](../../releases). 
Supercord will automatically keep itself up to date so you always have the latest features and patches!

## Building from source

Make sure you have [Bun](https://bun.sh/) installed.

```sh
# Install dependencies
bun install

# Build and start Supercord
bun run start

# Package into an installer
bun run package
```
